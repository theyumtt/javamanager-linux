# JavaManager

## ENG:
A modern, lightweight Graphical User Interface (GUI) for managing Java installations on Linux distributions. Built with Python and CustomTkinter, it simplifies the process of discovering, installing, and organizing various JDK and JRE versions.

## TUR:
Linux dağıtımlarında Java kurulumlarını yönetmek için modern, hafif bir Grafik Kullanıcı Arayüzü (GUI). Python ve CustomTkinter ile geliştirilen bu araç, çeşitli JDK ve JRE sürümlerini keşfetme, kurma ve düzenleme sürecini basitleştirir.
